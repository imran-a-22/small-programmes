Equivalent fractions name the same amount, even if the numbers look different.

Quick learning notes

  • Multiply top and bottom by the same number to make an equivalent fraction.
  • Example: 1/2 = 2/4 = 3/6.
  • If only one number changes, the value usually changes too.

How to play

  • Adjust angle and power.
  • Fire the glowing shot.
  • Hit the correct equivalent fraction chest.
